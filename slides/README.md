# Workshop Slides

Complete slide deck for the RAG workshop.

## Files

| File | Purpose |
|------|---------|
| `index.html` | **Ready to present** — open in any browser |
| `rag_workshop_slides.marp.md` | Editable source (Marp format) |
| `rag_workshop_slides.pptx` | PowerPoint export |

## Present immediately (no install)

```bash
open slides/index.html
```

Or double-click `index.html`.

**Controls:** arrow keys or space to advance, `F` for fullscreen, `S` for speaker notes (if supported).

## Edit and re-export with Marp (optional)

1. Install the [Marp for VS Code](https://marketplace.visualstudio.com/items?itemName=marp-team.marp-vscode) extension
2. Open `rag_workshop_slides.marp.md`
3. Export to PDF or HTML from the Marp panel

## Customize before presenting

Replace placeholders on slides 24-25:
- Your name and event name (title slide)
- Repo link
- Feedback form link
- Contact details
